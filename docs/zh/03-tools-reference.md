# 工具参考

本章列出 Plot 全部工具的绘制模式与可配置参数。激活工具后，右侧 **工具属性** 面板会显示对应选项。

---

## 基础工具

### 选择（Select）

| 模式 | 说明 |
|------|------|
| 普通选择 | 拖拽框选；**Ctrl+点击/框选** 向选择中添加或移除 |
| 套索选择 | 自由绘制选区轮廓 |

**框选方向：**

| 拖动方向 | 规则 |
|----------|------|
| 左->右 | 窗口选择：只选中完全包含在框内的图形 |
| 右->左 | 交叉选择：选中与框相交的图形 |

**其他操作：**

- 普通点击空白处：清除选择
- 普通点击图形：只选中该图形
- 选中单个图形后：可进入**控制点编辑**（拖动节点调整形状）
- 选择工具**不会**拖动图形位置；移动请用移动工具

**快捷键：** `Space`

### 橡皮擦（Eraser）

| 参数 | 范围 |
|------|------|
| 擦除半径 | 1–20 |

**快捷键：** `D`

## 绘图工具

### 直线（Line）

| 模式 | 说明 |
|------|------|
| 单线 | 点击起点和终点 |
| 多线 | 一次绘制多条平行线 |

多线模式额外参数：线条数量（2–20）、间距（5–100）。

**快捷键：** `L`

### 自由绘制（Free Draw）

| 参数 | 范围 |
|------|------|
| 平滑度 | 0.0–1.0 |

按住左键拖动绘制。平滑度越高，路径越光滑。

**快捷键：** `P`

### 圆形（Circle）

| 模式 | 说明 |
|------|------|
| 圆心-半径 | 先定圆心，再定半径 |
| 两点 | 直径两端确定圆 |
| 三点 | 圆周上三点确定圆 |

**快捷键：** `C`

### 矩形（Rectangle）

| 模式 | 说明 |
|------|------|
| 两点 | 对角两点 |
| 三点 | 一边加宽度方向 |
| 中心点 | 从中心向外扩展 |
| 圆角 | 带圆角的矩形 |

**快捷键：** `R`

### 椭圆（Ellipse）

| 模式 | 说明 |
|------|------|
| 三点定轴 | 长轴方向 + 短轴 |
| 三点定中心 | 中心 + 两轴端点 |
| 两点 | 外接矩形对角 |

**快捷键：** `E`

### 半圆（Semicircle）

| 模式 | 说明 |
|------|------|
| 两点 | 直径两端 |
| 三点 | 弧上三点 |

**快捷键：** `S`

### 圆弧（Arc）

| 模式 | 说明 |
|------|------|
| 起点-终点-方向 | 两点 + 弧方向 |
| 过三点 | 弧经过三个点 |
| 圆心-起点-终点 | 从圆心出发定弧 |

**快捷键：** `A`

### 折线（Polyline）

| 模式 | 说明 |
|------|------|
| 折线 | 逐段点击，双击或 Enter 结束 |
| 钢笔 | 锚点 + 曲线控制柄（贝塞尔） |
| 编辑 | 调整已有折线/曲线的节点与控制点 |

### 多边形（Polygon）

| 模式 | 说明 |
|------|------|
| 中心-顶点 | 中心 + 一个顶点定大小 |
| 中心-半径 | 中心 + 半径 |

额外参数：边数（滑块调节）。

### 样条曲线（Spline）

| 模式 | 说明 |
|------|------|
| 过点拟合 | 曲线经过每个点击点；张力 0–1 |
| 控制点 | 手动调整控制点 |

按 **C** 可闭合曲线。

### 悬链线（Catenary）

| 模式 | 说明 |
|------|------|
| 标准 | 对称下垂曲线 |
| 样条插值 | 非对称悬链形态 |

### 星形（Star）

| 参数 | 范围 |
|------|------|
| 顶点数 | 3–20 |
| 内旋角 / 外旋角 | 滑块调节 |

### 螺旋线（Spiral）

| 类型 | 说明 |
|------|------|
| 线性 | 等距螺旋 |
| 对数 | 指数增长螺旋 |
| 半圆 / 费马 / 斐波那契 / 多边形 | 各类型专属参数 |

通用选项：锐角边缘、边数、顺时针方向。

### 正弦曲线（Sine Wave）

| 参数 | 说明 |
|------|------|
| 相位 | 0–360°（有 0°/90°/180°/360° 预设按钮） |
| 振幅 / 波长 | 由鼠标拖拽确定 |

## 编辑工具

### 移动（Move）

选中图形后：

- **点-点模式**：点击基准点 -> 目标点
- **拖拽模式**：按住左键拖动图形

按住 **Shift** 可约束为水平/垂直方向移动。

### 旋转（Rotate）

| 参数 | 说明 |
|------|------|
| 角度步进 | 1–90°（配合 Shift 吸附） |
| 角度吸附 | 开关 |
| 预设角度 | 快捷按钮 |

按住 **Ctrl** 可临时复制后旋转。

### 缩放（Scale）

交互时点击确定缩放中心，拖拽控制缩放比例。

### 镜像（Mirror）

| 模式 | 说明 |
|------|------|
| 轴对称 | 两点定义镜像轴 |
| 中心对称 | 以一点为中心 180° 翻转 |

### 对齐（Align）

点选对齐：先选源点，再选目标点。可选 **允许缩放** 复选框。

### 阵列（Array）

| 模式 | 说明 |
|------|------|
| 矩形阵列 | 行数、列数、间距 |
| 环形阵列 | 数量、半径、角度步进 |
| 路径阵列 | 沿路径分布 |

激活后快捷键：**R** 矩形 / **C** 环形 / **P** 路径；方向键调整间距或数量。点击确认按钮完成。

### 偏移（Offset）

| 选项 | 说明 |
|------|------|
| 多重偏移 | 连续偏移生成多条平行线 |

### 圆角（Fillet / Round Corner）

| 参数 | 说明 |
|------|------|
| 圆角半径 | 滑块调节 |

在两条线交点处创建圆角。

### 倒角（Chamfer）

| 参数 | 说明 |
|------|------|
| 倒角距离 | 滑块调节 |

### 延伸（Extend）

自动延伸至最近边界，无需手动选择模式。

### 修剪（Trim）

| 模式 | 说明 |
|------|------|
| 边界修剪 | 以已有图形为修剪边界 |
| 围栏修剪 | 自定义围栏区域 |

围栏类型：折线、矩形、圆、椭圆、正多边形（3–24 边）。

| 参数 | 说明 |
|------|------|
| 容差 | 1–20 |
| Shift | 连续修剪 |

快捷键：**C** 点击修剪 / **F** 围栏修剪。

### 打断（Break）

| 模式 | 说明 |
|------|------|
| 单点打断 | 在一点处打断 |
| 两点打断 | 删除两点之间的部分 |

### 变换（Transform）

| 选项 | 说明 |
|------|------|
| 转角旋转 | 在折线转角处启用旋转 |

## 标注工具

### 文字（Text）

| 参数 | 说明 |
|------|------|
| 字号 | 滑块 |
| 粗体 / 斜体 | 复选框 |
| 对齐方式 | 左 / 中 / 右 |
| 行高 | 滑块 |
| 输入方式 | 对话框 / 内联 |
| 转形状 | 将文字转为矢量图形 |

### 标注（Annotation）

| 模式 | 说明 |
|------|------|
| 距离 | 测量两点距离 |
| 角度 | 测量角度 |
| 半径 | 测量圆弧/圆半径 |
| 面积 | 测量封闭区域面积 |

## 工具选项通用技巧

1. **Shift** — 几乎所有绘图工具都支持正交约束（水平/垂直/45° 等）
2. **吸附** — 配合端点、中点、交点吸附可精确定位；见 [设置与快捷键](05-settings-shortcuts.md)
3. **工具属性面板** — 切换模式后，操作方法提示会实时更新
4. **撤销** — 所有绘图与编辑操作均支持 `Ctrl+Z` 撤销
