# 常见问题

## 操作相关

### Esc 为什么不能关闭 Plot？

`Esc` 在 Plot 中的优先级是：

1. 取消当前工具正在进行的操作
2. 清除图形选择
3. 清除幽灵方块预览

要退出 Plot，请点击右上角的 **关闭按钮（✕）**。

### 为什么点击画布没有反应？

鼠标悬停在面板（工具栏、属性面板等）上时，输入由 ImGui 面板捕获。请将光标移到 **画布中央区域**（面板之间的空白区域）再操作。

### 线转方块按钮是灰色的

必须先 **选中** 至少一个图形。按 `Space` 切换到选择工具，框选或点击目标图形后再试。

### 投影方块没有效果

投影需要先有幽灵方块预览。请按以下顺序操作：

1. 选中图形
2. 线转方块（生成预览）
3. 投影方块（放置）

### 如何删除图形？

- 选中后按 `Delete`
- 或使用 **橡皮擦** 工具
- 所有删除操作均可 `Ctrl+Z` 撤销

## 快捷键相关

### 快捷键不生效

1. 确认没有输入框处于焦点状态（如文字工具、搜索框）
2. 检查是否与其他模组快捷键冲突
3. 打开 **设置与帮助 → 快捷键** 确认绑定是否正确
4. 尝试 **恢复默认** 后重新绑定

### P 键是做什么的？

`P` 是 Plot **内部**自由绘制工具的默认快捷键，与打开 Plot 的按键（默认 `0`）无关。仅在 Plot 界面打开时生效。

### 如何修改「打开 Plot」的按键？

在 Minecraft **设置 → 控制 → Plot** 分类中修改，不在 Plot 内的快捷键设置中。

## 视图相关

### 如何缩放和平移？

| 操作 | 功能 |
|------|------|
| 滚轮 | 缩放 |
| 中键拖拽 | 平移 |
| 视图范围滑块 | 调整可视范围 |

### 视图被锁定了

点击顶部 **锁定视图** 按钮解锁。锁定期间视图范围滑块不可用。

### 如何切换回正常游戏视角？

- 点击 **相机切换** 按钮切换到透视模式
- 或关闭 Plot 界面（自动恢复）

## 方块建造相关

### 幽灵方块是什么？

半透明预览方块，不是真实方块。用于在放置前确认设计效果。可用 Esc 清除或关闭 Plot 时自动清除。

### 投影的方块可以撤销吗？

可以。投影操作记录在命令历史中，按 `Ctrl+Z` 即可撤销。

### 调色板最多几个方块？

最多 **14** 个槽位。

### 精简转换和完整转换有什么区别？

| 模式 | 适用场景 |
|------|----------|
| 完整转换 | 需要精确还原每个细节 |
| 精简转换 | 大型结构，减少方块数量；可通过精简比率调节简化程度 |

## 功能限制

### 项目能保存吗？

当前版本的工具栏 **没有** 完整的项目保存/加载按钮。以下数据会自动持久化：

- 快捷键（`plot/keymap.json`）
- 工具参数（关闭 Plot 时保存）
- 插件配置与插件项目数据
- **图库条目**（用户保存的图形）

画布上的图层内容在关闭 Plot 后 **不会** 自动保存为项目文件。可将常用图形通过 [图库](07-gallery.md) 保存后复用。

### 扩展面板和插件在哪里？

在右侧栏切换到 **扩展** 标签页，点击插件图标并勾选 **启用插件**。详见 [扩展插件](08-plugins.md)。

内置插件包括：**道路系统**、**土方平衡**、**建筑轮廓生成器**。

### 哪些工具在语言文件中出现但无法使用？

`填充（Fill）` 和 `拉伸（Stretch）` 在语言文件中有条目，但尚未注册到工具面板，暂时无法使用。

## 性能与兼容性

### 与其他 ImGui 模组冲突？

Plot 使用 imgui-java 1.86.12，已修复多 ImGui 模组共存时的上下文冲突问题。若仍有异常，请检查是否安装了多个 ImGui 模组。

### 方块图标加载慢？

Plot 会在进入世界后预加载常用方块图标缓存。首次打开方块配置可能稍慢，之后会明显改善。

### 游戏会暂停吗？

不会。Plot 设计器打开时游戏继续运行，怪物仍会移动。建议在安全区域使用。
