# 界面介绍

Plot 设计器采用可停靠的 ImGui 面板布局，画布铺满全屏，面板浮动其上。

## 整体布局

```
┌──────────────────────────────────────────────────────────────────┐
│  控制面板（顶部）                              系统面板（右上）    │
│  撤销/吸附/方块/视图…                          主题切换 + 关闭 ✕  │
├──────────┬───────────────────────────────────────┬─────────────┤
│ 工具面板  │         画布（全屏，世界背景可见）        │  属性面板    │
│ （左侧）  │         在此区域绘制与编辑图形            │  （右侧）    │
│          │                                       │             │
└──────────┴───────────────────────────────────────┴─────────────┘
```

所有面板均支持 **拖拽与停靠**（Docking），你可以按需调整布局。首次打开时会自动生成默认布局。

> **提示：** 只有画布中央区域会接收绘图点击；鼠标悬停在面板上时，输入由面板捕获。

## 顶部控制面板

从左到右依次为：

### Plot 标志

点击打开 **设置与帮助** 对话框，包含快捷键、吸附反馈、内置帮助等。详见 [设置与快捷键](05-settings-shortcuts.md)。

### 文件工具

| 按钮 | 功能 |
|------|------|
| 撤销 | 撤销上一步操作 |
| 重做 | 重做已撤销的操作 |

### 工具设置

| 按钮 | 左键 | 右键 |
|------|------|------|
| **吸附** | 开关吸附 | 打开吸附设置对话框 |
| **网格** | 开关网格 | 打开网格设置对话框 |
| **清除** | 清空整个画布（可撤销） | — |

### 方块操作

| 按钮 | 左键 | 右键 |
|------|------|------|
| **方块配置** | 打开方块选择界面 | — |
| **线转方块** | 将选中图形转为幽灵方块预览 | 打开线转方块设置 |
| **投影方块** | 将幽灵方块放置到世界 | 打开投影设置 |

> 未选中任何图形时，**线转方块** 按钮为灰色不可用。

### 视图工具

| 按钮 | 左键 | 右键 |
|------|------|------|
| **相机切换** | 正交 ↔ 透视 切换 | 打开正交相机设置 |
| **锁定视图** | 锁定/解锁画布平移与缩放 | — |

### 滑块控制

| 滑块 | 范围 | 说明 |
|------|------|------|
| **视图范围** | 40–310 | 视图锁定时不可用；右键可输入精确数值 |
| **画布透明度** | 0–100% | 控制绘图层相对世界的透明度 |

## 左侧工具面板

按分组排列所有工具，组间以分隔线区分：

| 分组 | 包含工具 |
|------|----------|
| 基础工具 | 选择、橡皮擦 |
| 绘图工具 | 直线、自由绘制、圆、矩形、样条、椭圆、半圆、圆弧、折线、多边形 |
| 特殊图形 | 星形、螺旋、悬链线、正弦曲线 |
| 编辑工具 | 移动、旋转、缩放、镜像、对齐、阵列、偏移、打断、圆角、倒角、延伸、修剪、变换 |
| 标注工具 | 文字、标注 |

点击工具图标即可激活。当前工具的属性会显示在右侧 **工具属性** 区域。

## 右侧属性面板

属性面板包含多个可折叠区域：

### 工具属性

显示当前激活工具的：
- 名称与描述
- 绘制模式 / 参数选项
- 使用方法提示（操作时会被实时状态信息覆盖）

### 图层

管理绘图图层，支持：

| 操作 | 说明 |
|------|------|
| 新建图层 | 弹出命名对话框 |
| 删除 | 需先选中；不能删除最后一个或已锁定的图层 |
| 合并 | 将 2 个及以上选中图层合并为「合并图层」 |
| 上移 / 下移 | 调整图层叠放顺序 |
| 全选图层内容 | 选中当前图层所有图形 |

每个图层行包含：
- **可见性**（眼睛图标）
- **锁定**（锁图标）
- **名称**（可编辑）
- **颜色 / 线型**
- 右键 **上下文菜单**
- **拖拽** 调整顺序

### 历史记录

显示最近 30 条操作记录，带时间戳。点击某条记录可跳转到对应状态（自动执行撤销或重做）。

### 状态属性

显示当前画布与选择的统计信息。

## 对话框一览

除属性面板外，Plot 还通过弹出对话框进行详细设置：

| 对话框 | 打开方式 |
|--------|----------|
| 设置与帮助 | 点击 Plot 标志 |
| 吸附设置 | 右键顶部「吸附」按钮 |
| 网格设置 | 右键顶部「网格」按钮 |
| 线转方块设置 | 右键顶部「线转方块」按钮 |
| 方块投影设置 | 右键顶部「投影方块」按钮 |
| 正交相机设置 | 右键顶部「相机切换」按钮 |
| 方块配置 | 左键顶部「方块配置」按钮（原生 Minecraft UI） |

## 通用修饰键

在绘图和编辑过程中，以下按键贯穿多个工具：

| 按键 | 作用 |
|------|------|
| **Shift**（按住） | 正交/角度约束；部分工具中可临时禁用吸附 |
| **Ctrl**（按住） | 部分变换工具中临时复制（如旋转） |
| **Delete** | 删除选中图形（可撤销） |
| **Esc** | 取消当前工具操作 → 清除选择 → 清除幽灵方块预览 |
| **Ctrl+Z** | 撤销 |
| **Ctrl+Y** / **Ctrl+Shift+Z** | 重做 |
