# 扩展插件

Plot 内置多个插件，通过右侧栏的 **扩展** 标签页访问。

## 打开扩展面板

1. 在 Plot 界面右侧切换到 **扩展** 标签
2. 在顶部图标栏中点击要使用的插件
3. 勾选 **启用插件** 后，下方会显示该插件的参数与操作界面

> 插件默认可能未启用，需手动勾选后才能使用。

## 已安装插件

| 插件 | 说明 |
|------|------|
| **土方平衡** | 计算和优化场地平整时的土方挖填方案 |
| **道路系统** | 沿路径规划与生成道路结构 |
| **建筑轮廓生成器** | 从多边形或矩形轮廓生成建筑 |

---

## 道路系统

### 典型流程

1. 在画布上用折线、样条等工具绘制道路中心线
2. 打开 **扩展** -> 选择 **道路系统** -> 启用插件
3. 在插件面板中认领路径、设置路宽与材质
4. 生成幽灵方块预览，确认后投影到世界

### 主要能力

- 道路预设（城市主干道、次干道、乡村公路、高速公路等）
- 路宽、材质、坡度与桥梁/隧道阈值
- 人行道、路肩、排水沟等附属结构
- 挖填方估算与预览建造

配置文件：`.minecraft/config/plugins/road_system.json`

---

## 土方平衡

### 典型流程

1. 绘制或选择封闭区域作为整平范围
2. 打开 **扩展** -> **土方平衡** -> 启用插件
3. 在 **认领** 标签页拾取/认领区域
4. 在 **编辑** 标签页设置整平方式（水平、定坡、三点定面、拟合斜面等）
5. 在 **生成** 标签页预览挖填方并建造

### 主要能力

- 多种整平曲面模式
- 挖方/填方材质与系数设置
- 区域列表管理与撤销/重做
- 幽灵方块预览（挖方/填方分色）

---

## 建筑轮廓生成器

### 典型流程

1. 在画布上绘制建筑底面轮廓（矩形或多边形）
2. 打开 **扩展** -> **建筑轮廓生成器** -> 启用插件
3. 拾取/认领轮廓
4. 配置楼层、高度等参数
5. 生成预览并投影建造

### 主要能力

- 从 2D 轮廓生成 3D 建筑体块
- 多建筑项目管理
- 幽灵预览与实际方块放置

## 插件数据保存

各插件会在 `.minecraft/plot/plugins/` 下保存自己的项目数据（JSON）。插件配置保存在 `config/plugins/` 目录。

> 注意：这与画布主设计的「项目文件」不同。当前版本画布图层尚无统一的保存/加载 UI，但插件各自的数据会按会话自动持久化。

## 常见问题

### 扩展面板是空的或按钮无反应

确认已在插件名称旁勾选 **启用插件**。

### 拾取区域时没有响应

拾取模式会临时接管画布点击。完成拾取前，请勿切换到其他工具；按 **Esc** 可取消拾取。

### 与绘图工具冲突

土方/建筑的区域拾取、图库放置一样，在特殊模式下会优先捕获画布输入。留意状态栏提示。
