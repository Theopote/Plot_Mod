# 图库

图库面板用于浏览预设图形、保存自己的设计片段，并快速放置到画布上。在没有完整项目保存功能时，图库也是保存常用图形的实用方式。

## 打开图库

在右侧栏切换到 **图库** 标签页（与 **属性**、**扩展** 并列）。

## 界面说明

| 区域 | 功能 |
|------|------|
| 搜索框 | 按名称或描述过滤条目 |
| 分类列表 | 全部 / 建筑 / 景观 / 图形，以及自定义分类 |
| 条目表格 | 名称、描述、操作按钮 |
| 工具栏 | 保存选中到图库、增加分类 |

## 内置预设

图库自带若干预设，例如：

- **建筑**：教堂平面、城堡平面、古典别墅、合院平面
- **景观**：景观水池
- **图形**：矩形块、圆形块、三角形、泰森多边形镶嵌等

内置预设为只读，不可编辑或删除。

## 保存选中图形

1. 用选择工具选中画布上的一个或多个图形
2. 打开 **图库** 面板
3. 点击 **保存选中到图库**
4. 在对话框中填写名称、描述，选择分类后确认

保存的条目会写入本地图库数据，下次打开 Plot 仍可使用。

## 放置到画布

1. 在图库列表中找到目标条目
2. 点击 **在画布中心打开**（或同类放置按钮）
3. 在画布上点击以确定放置位置
4. 按 **Esc** 或点击取消可退出放置模式

> 图库放置模式激活时，绘图工具不会接收点击，避免误操作。

## 管理分类

| 操作 | 说明 |
|------|------|
| 增加分类 | 创建自定义分类 |
| 重命名分类 | 右键或上下文菜单（自定义分类） |
| 删除分类 | 需确认；内置分类不可删 |

## 删除条目

对用户保存的条目，可通过操作列的删除按钮移除。删除前会弹出确认对话框。

## 技巧

- 把重复使用的结构（门洞、标准房间、道路截面等）存入图库，比每次重画更高效
- 配合图层使用：从图库放置后，可移到专用图层统一管理
- 图库保存的是图形数据，不包含方块配置；放置后仍需单独做线转方块与投影
