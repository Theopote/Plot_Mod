# 设置与快捷键

Plot 提供丰富的自定义选项，涵盖快捷键、吸附、网格和视觉反馈。

---

## 打开设置

点击顶部控制面板最左侧的 **Plot 标志**，打开 **设置与帮助** 对话框。

对话框包含三个标签页：

| 标签页 | 内容 |
|--------|------|
| 快捷键 | 自定义工具切换与编辑快捷键 |
| 吸附与反馈 | 吸附标记显示与颜色 |
| 帮助与教程 | 与 docs/ 对应的 8 章内置指南（快速入门、界面、工具、方块建造、设置、图库、插件、FAQ） |

---

## 快捷键

### Minecraft 按键绑定

在 **设置 → 控制 → Plot** 分类中：

| 按键 | 功能 | 默认值 |
|------|------|--------|
| 打开 Plot 界面 | 进入/退出设计器入口 | `0` |

### Plot 内快捷键（可自定义）

在 **设置与帮助 → 快捷键** 标签页中管理。配置文件保存在：

```
.minecraft/plot/keymap.json
```

#### 默认绑定

| 功能 | 默认按键 | 分类 |
|------|----------|------|
| 选择 | `Space` | 绘图工具 |
| 橡皮擦 | `D` | 绘图工具 |
| 直线 | `L` | 绘图工具 |
| 自由绘制 | `P` | 绘图工具 |
| 圆形 | `C` | 绘图工具 |
| 矩形 | `R` | 绘图工具 |
| 椭圆 | `E` | 绘图工具 |
| 半圆 | `S` | 绘图工具 |
| 圆弧 | `A` | 绘图工具 |
| 撤销 | `Ctrl+Z` | 编辑操作 |
| 重做 | `Ctrl+Y` | 编辑操作 |

#### 自定义方法

1. 打开 **设置与帮助 → 快捷键**
2. 使用搜索框快速定位
3. **双击** 某行或点击 **编辑** 进入录制模式
4. 按下想要的按键组合
5. 点击 **完成**

| 操作 | 说明 |
|------|------|
| **Backspace** | 清除当前绑定 |
| **Esc** | 取消录制 |
| **恢复默认** | 重置所有快捷键 |

> **说明：** 打开 Plot 默认使用数字键 `0`，与 Plot 内自由绘制工具（`P`）分开，避免冲突。可在 Minecraft 控制设置中改为其他按键。

> **覆盖范围：** Keymap 仅包含常用绘图工具与撤销/重做。Spline、Polyline、Move 等工具暂无默认快捷键，需从左侧工具栏点击激活，或在 Keymap 中自行扩展（若已注册）。

### 全局快捷键（不可在 Keymap 中配置）

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+A` | 全选所有可见图层中的图形 |
| `Delete` | 删除选中图形 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` / `Ctrl+Shift+Z` | 重做 |
| `Esc` | 取消操作 → 清除选择 → 清除幽灵方块 |

## 吸附设置

### 快速开关

- 左键顶部 **吸附** 按钮：全局开关
- 右键顶部 **吸附** 按钮：打开详细设置对话框

### 几何特征吸附

| 类型 | 说明 |
|------|------|
| 端点 | 线段端点 |
| 中点 | 线段中点 |
| 圆心 | 圆/弧圆心 |
| 重心 | 封闭图形重心 |
| 顶点 | 多边形顶点 |
| 象限点 | 圆/椭圆的 0°/90°/180°/270° 点 |
| 网格点 | 网格交叉点 |
| 垂足 | 垂直投影点 |
| 交点 | 两图形交点 |
| 最近点 | 图形上距光标最近的点 |
| 控制点 | 样条/贝塞尔控制点 |
| 切点 | 曲线切线方向点 |

### 几何关系约束

| 类型 | 说明 |
|------|------|
| 水平 | 限制为水平方向 |
| 垂直 | 限制为竖直方向 |
| 平行 | 与参考线平行 |
| 延伸 | 沿已有线段方向延伸 |

### 其他吸附选项

| 选项 | 说明 |
|------|------|
| 吸附半径 | 像素或毫米单位（Alt 切换单位） |
| 标记大小 | 2–10 px |
| 吸附级别 | 全局 / 工具 / 图层 |
| 优先级 | 按类型优先 或 按距离优先 |
| 排除隐藏图层 | 不对隐藏图层上的图形吸附 |
| Shift 临时禁用 | 按住 Shift 时暂停吸附 |
| 标记动画 | 吸附命中时的视觉反馈 |

### 吸附与反馈（设置对话框）

在 **设置与帮助 → 吸附与反馈** 标签页中：

- 开关各类吸附标记的显示
- 自定义各类型吸附标记的 **颜色**
- 控制点显示 / 点索引显示
- **重置所有颜色**

## 网格设置

右键顶部 **网格** 按钮打开网格设置：

| 参数 | 范围 |
|------|------|
| 网格大小 | 8–64 |
| 透明度 | 滑块 |
| 线宽 | 滑块 |
| 颜色 | 颜色选择器 |

左键 **网格** 按钮可快速开关网格显示。

---

## 正交相机设置

右键顶部 **相机切换** 按钮：

| 参数 | 范围 |
|------|------|
| 缩放比例 | 0.1–10 |
| 视距 | 0–100 |
| 近裁剪面 / 远裁剪面 | 数值输入 |
| 重置默认 | 一键恢复 |

## 持久化配置

Plot 会在关闭时自动保存以下配置：

| 文件 | 内容 |
|------|------|
| `plot/keymap.json` | 快捷键绑定 |
| 工具配置 | 各工具的绘制模式与参数 |
| `config/plugins/*.json` | 插件配置（如道路系统） |

> **注意：** 当前版本 **没有** 项目文件保存/加载的 UI 入口。画布内容在关闭 Plot 后不会自动保存为项目文件。操作历史仅在当前会话内有效。
