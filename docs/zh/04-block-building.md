# 方块建造

Plot 的核心建造流程是：在画布上绘制设计 -> 配置方块 -> 生成预览 -> 投影到世界。本章详细说明每一步。

---

## 整体流程

```
绘制图形 -> 选中目标 -> 方块配置 -> 线转方块（预览）-> 调整投影设置 -> 投影方块（放置）
```

---

## 方块配置

方块配置界面是原生 Minecraft UI（非 ImGui 面板），用于选择建造时使用的方块。

### 打开方式

点击顶部控制面板的 **方块配置** 按钮。

### 界面说明

| 区域 | 功能 |
|------|------|
| 左侧分类栏 | 按方块类别浏览 |
| 搜索框 | 按名称或命名空间搜索 |
| 方块网格 | 分页显示方块，点击选择 |
| 调色板 | 已选方块列表（最多 **14** 个槽位） |

### 操作

| 操作 | 效果 |
|------|------|
| 左键点击方块 | 添加到调色板 |
| 右键点击方块 | 从调色板移除 |
| 左键点击调色板槽位 | 调整排列顺序 |
| **应用** | 保存配置 |
| **取消** | 放弃更改 |
| **清空** | 清除调色板 |

> 线转方块会按调色板中的方块顺序映射到不同的线条或区域。

---

## 线转方块

将选中的矢量图形转换为世界中的 **幽灵方块预览**（半透明，非实体方块）。

### 前提条件

- 必须 **先选中** 一个或多个图形
- 建议先完成 **方块配置**

### 设置（右键「线转方块」按钮）

| 选项 | 说明 |
|------|------|
| **转换模式** | |
| └ 完整转换 | 忠实还原所有细节 |
| └ 精简转换 | 简化方块数量，适合大型结构 |
| **精简比率** | 0.1–1.0（仅精简模式） |
| **封闭填充** | 封闭图形是否填充内部（否则仅描边） |

### 执行

1. 选中目标图形
2. （可选）右键调整设置
3. 左键点击 **线转方块**
4. 在世界中查看半透明幽灵方块预览

> 高度默认取玩家脚下标高。预览可随时用 **Esc** 清除。

---

## 方块投影

将幽灵方块预览 **放置为真实方块**。

### 设置（右键「投影方块」按钮）

| 模式 | 说明 |
|------|------|
| 投影到地面 | 自动贴合地形表面（默认） |
| 投影到指定标高 | 固定 Y 坐标（-64 至 320） |

### 执行

1. 确保已有幽灵方块预览（先执行线转方块）
2. （可选）右键调整投影设置
3. 左键点击 **投影方块**
4. 方块被放置到世界中

> 投影操作支持 **撤销**（`Ctrl+Z`）。

---

## 完整示例：绘制一堵墙

### 步骤

1. **绘制** — 选择矩形工具（`R`），绘制一面墙的轮廓
2. **选择** — 按 `Space`，框选刚绘制的矩形
3. **配置方块** — 打开方块配置，将「石砖」加入调色板，点击应用
4. **预览** — 左键点击线转方块，在世界中查看幽灵预览
5. **调整** — 若高度不对，右键投影设置，选择「投影到指定标高」并设置 Y 值
6. **放置** — 左键点击投影方块，墙被建造到世界中
7. **撤销**（如需）— `Ctrl+Z` 可撤销投影

### 技巧

- 对复杂设计，建议分层绘制（不同图层对应不同结构部位）
- 精简转换适合长距离道路或大型围墙，可显著减少方块数量
- 投影前先用正交相机确认设计位置，必要时平移画布对齐

---

## 幽灵方块说明

| 特性 | 说明 |
|------|------|
| 外观 | 半透明，可透视 |
| 性质 | 非实体，不影响碰撞 |
| 清除方式 | 按 Esc、关闭 Plot、或重新线转方块 |
| 与投影关系 | 必须先有幽灵预览才能投影 |
