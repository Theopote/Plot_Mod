# 快速入门

Plot 是一款 Minecraft Fabric 模组，提供类似 CAD 的 2D 规划工具。你可以在正交俯视视角下绘制几何图形，再将设计投影为方块，直接在世界里建造。

## 环境要求

| 项目 | 版本 |
|------|------|
| Minecraft | 1.21.10 |
| Fabric Loader | ≥ 0.18.4 |
| Fabric API | 最新兼容版 |
| Java | ≥ 21 |

## 安装步骤

1. 从 [Fabric 官网](https://fabricmc.net/use/) 安装对应版本的 Fabric Loader
2. 下载 [Fabric API](https://modrinth.com/mod/fabric-api) 并放入 `mods` 文件夹
3. 将 Plot 模组 JAR 放入 `.minecraft/mods` 目录
4. 启动游戏

## 打开 Plot 设计器

有两种方式进入 Plot 界面：

| 方式 | 操作 |
|------|------|
| **快捷键** | 在游戏中按 **`0`**（可在 设置 → 控制 → Plot 中修改） |
| **物品** | 在创造模式 **工具** 物品栏中找到 **Plot** 物品，右键使用 |

> 必须在已进入世界且控制角色的状态下才能打开 Plot。

## 打开后会发生什么

- 自动切换到 **正交俯视相机**，方便平面绘图
- 隐藏 HUD（物品栏、玩家模型等），但 **游戏不会暂停**
- 背景仍显示 Minecraft 世界，画布以半透明叠加其上
- 首次打开时会初始化 UI 界面（ImGui）

## 关闭 Plot

点击右上角的 **关闭按钮（✕）** 退出设计器。

> **注意：** `Esc` 键 **不会** 关闭 Plot 界面，只会取消当前操作或清除选择。详见 [常见问题](06-faq.md)。

关闭时会自动：
- 恢复透视相机与 HUD
- 清除幽灵方块预览
- 保存工具配置

## 五分钟上手

### 1. 绘制一条直线

1. 按 `L` 或点击左侧工具栏的 **直线** 工具
2. 在画布上点击起点，再点击终点
3. 按住 **Shift** 可约束为水平/垂直方向

### 2. 选择图形

1. 按 `Space` 切换到 **选择** 工具
2. 拖拽框选多个图形（左→右为窗口选择，右→左为交叉选择）
3. 按住 **Ctrl** 再点击或框选，可向选择中添加或移除图形

> 选择工具**不会**拖动图形。移动请使用移动工具。

### 3. 移动图形

1. 保持选中状态，切换到 **移动** 工具
2. 点击基准点，再点击目标位置（或直接拖拽移动）

### 4. 撤销操作

- 点击顶部工具栏的 **撤销** 按钮
- 或按 `Ctrl+Z`

### 5. 将图形变成方块

1. 选中要转换的图形
2. 点击 **方块配置**，选择建造用方块并应用
3. 点击 **线转方块** 生成幽灵方块预览
4. 点击 **投影方块** 将预览放置到世界中

完整流程见 [方块建造](04-block-building.md)。

## 画布导航

| 操作 | 功能 |
|------|------|
| **滚轮** | 缩放视图 |
| **中键拖拽** | 平移画布 |
| **顶部「视图范围」滑块** | 调整可视范围（40–310） |
| **顶部「画布透明度」滑块** | 调整绘图叠加层的透明度（0–100%） |
